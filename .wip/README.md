# pi-fiction WIP

A lightweight workshop board for framework ideas while `pi-fiction` takes shape.

Ideas live in `ideas/` and use:

```text
{topic_slug}-{issue_number}-{name}.md
```

Done or abandoned ideas can move to `ideas/.archive/` during cleanup.

## Ideas, tasks, and plans

- **Ideas** in `.wip/ideas/` shape creative framework concepts before the deliverable is clear.
- **Tasks** in `.wip/tasks/` describe accepted work with a concrete deliverable.
- **Plans** keep the normal `pi-code` meaning: implementation plans for tasks.

A framework idea can produce one or more tasks once its shape is clear. If the work still needs discovery, keep it as an idea.
